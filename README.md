# 🏥 Student Health Tracker System

## 📌 Project Overview

The **Student Health Tracker System** is a Java-based desktop application designed to help manage and monitor student health information.
The system allows administrators to record and manage student health data, while students can view their health records and receive health suggestions.

This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts such as classes, objects, encapsulation, and modular design while building a practical healthcare monitoring application.

---

## 🎯 Objectives

* To maintain student health records in an organized system
* To track important health parameters such as BMI
* To provide health suggestions based on student records
* To demonstrate Java GUI-based application development

---

## 🚀 Features

### 👨‍⚕️ Admin Dashboard

* Add and manage student profiles
* View student health records
* Maintain medical information
* Manage doctor contact information

### 🎓 Student Dashboard

* View personal health records
* Check BMI and health information
* Access doctor contact details

### 📊 Health Tracking

* Record health parameters
* Calculate BMI automatically
* Store and manage student health data
